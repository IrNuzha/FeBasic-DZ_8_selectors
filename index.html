Задание №1

Какими бывают селекторы:

  • Селектор элемента - предназначен для выбора элементов по имени тега
  • Селектор по классу - предназначен для выбора элементов по классу (значению атрибута class)
  • Селектор по идентификатору (id) - предназначен для выбора элемента по идентификатору (значению атрибута id)
  • Универсальный селектор - предназначен для выбора всех элементов (*)
  • Псевдоклассы - предназначены для более точного выбора элементов в зависимости от их динамического состояния или расположения
  • Селектор по нескольким классам - предназначен для выбора элемента, имеющего одновременно два класса, не затронув элементы с одним из этих классов
  • Селектор по вложенности - предназначен для возможности задать стиль для тега только когда он располагается внутри определенного контейнера
  • Дочерние селекторы - предназначен для элемента, который непосредственно располагается внутри родительского элемента
  • Селекторы атрибутов - устанавливает стиль для элемента, если задан специфичный атрибут тега
  • Родственные селекторы - правила применяются ко всем близлежащим элемента, которые имеют общего родителя
